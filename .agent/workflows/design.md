# Workflow: Visual Alchemy ("The Designer")

> **Objective**: Create a visual experience that feels "Inevitably Premium".
> **Philosophy**: "Good design is obvious. Great design is invisible."

## Phase 1: Evidence Visualization (margin-visualizer)

1.  **Data Extraction**:
    *   Identify every number in `index.html`.
    *   *Decision*: If it's a key stat, it MUST be a chart, not text.

2.  **SVG Architecture**:
    *   **Action**: `margin-visualizer` generates bespoke SVGs.
    *   **Constraint**: NO external libraries (Chart.js/D3). Pure SVG paths.
    *   **Style**: Apply "Neon Glow" filters, gradients, and match the theme colors.

## Phase 2: Atmospheric Design (margin-designer)

3.  **Theme Injection**:
    *   **Action**: Apply the user-selected theme (e.g., `theme-swiss`) to `<body>`.
    *   **Math Check**: Ensure all margins/padding follow the Fibonacci scale.

4.  **Topological Betrayal**:
    *   **Action**: Break the grid.
    *   *Technique*: Offset the image 20% off-canvas. Overlap the headline with the chart.
    *   *Goal*: Stop the user's scanning eye to force engagement.

5.  **Atmospherics**:
    *   **Action**: Add noise, vignettes, glassmorphism layers.
    *   *Why*: To separate "Web Page" from "Cinematic Experience".

## Success Criteria (The "1% Check")

*   [ ] Are there any "plain" straight lines? (Everything should have easing/texture).
*   [ ] Do the charts look like they belong to *this specific* brand?
*   [ ] Does it export cleanly to PDF without cutting off content?

## Usage
Run this workflow by instructing the agent:
`"Apply the design to the current draft."`
