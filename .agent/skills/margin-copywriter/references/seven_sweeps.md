# The Seven Sweeps (Editing Protocol)

> "Write drunk, edit sober."

Do not try to fix everything at once. Do 7 passes, looking for ONE thing each time.

1.  **The Logic Sweep**: Does the argument make sense? Is there a hole in the plot?
2.  **The Clarity Sweep**: Are there any sentences I have to read twice?
3.  **The Rhythm Sweep**: Read it out loud. Does it stumble? (Look for unintentional rhymes or clunky alliteration).
4.  **The Slop Sweep**: Ctrl+F for banned words ("delve", "leverage").
5.  **The Adverb Sweep**: Kill all words ending in "-ly".
6.  **The "You" Sweep**: Count the "We" vs "You". "You" must win 80/20.
7.  **The Headline Sweep**: Can I make the headers 30% shorter?
