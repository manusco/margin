# The Source Triangulation Protocol

> "One source is a rumor. Two is a coincidence. Three is a fact."

## The Hierarchy of Evidence

1.  **Platinum Tier (Immutable Truth)**
    *   Source Code (GitHub Repo).
    *   Raw Logs / Database Dumps.
    *   Video Evidence (Unedited).

2.  **Gold Tier (High Trust)**
    *   Peer-Reviewed Whitepapers.
    *   Official Engineering Blogs form Reputable Companies (Netflix, Uber, Google).
    *   Verified creator (Known Expert).

3.  **Silver Tier (Trust but Verify)**
    *   StackOverflow Accepted Answers.
    *   Mainstream Tech News (TechCrunch, etc.).
    *   Documentation (often outdated).

4.  **Garbage Tier (Ignore)**
    *   "Top 10 Tools" SEO Spam.
    *   LinkedIn Thought Leadership posts without data.
    *   ChatGPT unverified output.

## The Triangulation Rule
If a claim is controversial or critical (e.g., "This library is deprecated"), you must find **3 independent sources** from Gold or Platinum tiers.

*   **Source A**: The GitHub Issue tracker says it's broken.
*   **Source B**: A chaotic blog post mentions a workaround.
*   **Source C**: The release notes confirm the breaking change.
