# Workflow: Initialize Project ("The Big Bang")

> **Objective**: Set up a pristine environment for a new Lead Magnet.

## 1. Scaffold (margin-librarian)
1.  **Ask**: "What is the project slug?" (e.g. `pitch-deck-q1`).
2.  **Copy**: Duplicate `projects/_template` to `projects/[slug]`.
3.  **Clean**: Reset the `index.html` to a blank slate (keep only the cover).

## 2. Strategy Setup (margin-analyst)
1.  **Create**: Generate a blank `.margin/brief_template.md` in the new folder.
2.  **Prompt**: "The folder is ready. Shall I run the research workflow now?"

## Usage
`"Initialize a new project called 'q1-investor-deck'."`
