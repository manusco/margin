# Contributing to MARGIN

Thank you for helping us build the standard for Agentic Publishing.

## How to Help

### 1. Improve the "Brain" (Skills & References)
The core value of MARGIN is its "1% Knowledge".
*   **Add a Protocol**: Do you have a better way to structure arguments? Add a file to `.agent/skills/margin-analyst/references/`.
*   **Refine a Skill**: Edit `SKILL.md` if you find an agent is hallucinating or being lazy.

### 2. Improve the "Engine" (Framework)
*   **Optimize Paged.js**: We always need better print layout handling.
*   **New Themes**: Submit a Pull Request with a new preset in `.margin/theme-presets.json`.

---

## The Golden Rule for PRs
**"If it looks generic, reject it."**
MARGIN exists to fight the "Template Fatigue" of modern tools. Any contribution must raise the aesthetic or intellectual bar.

## Development Setup
1.  Clone repo.
2.  `npm install`
3.  `npm run dev`
